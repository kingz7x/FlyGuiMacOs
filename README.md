# 🍌 Bananao Hub v2.0

A modern Roblox utility hub featuring a sleek interface, Fly Hack, NoClip, player management tools, window resizing, and keyboard shortcuts.

## Features

### ✈️ Fly Hack

* Enable/Disable fly mode instantly.
* Adjustable flight speed (10 - 300).
* Smooth camera-based movement.
* Controls:

  * **W/A/S/D** → Move
  * **Space** → Fly Up
  * **Left Ctrl** or **C** → Fly Down

### 🚪 NoClip

* Walk through walls and obstacles.
* Toggle on/off at any time.

### 📍 Quick Teleport

* Teleport up by 50 studs.
* Teleport down by 50 studs.

### 🖥️ Modern Interface

* Inspired by modern desktop operating systems.
* Draggable window.
* Resizable window.
* Smooth animations and transitions.
* Traffic-light style controls.

### ⌨️ Keyboard Shortcut

* Press **ALT** to show or hide the UI instantly.

### 🎨 Custom UI

* Elegant gray theme.
* Rounded corners.
* Gradient backgrounds.
* Modern card-based layout.

## Installation

Execute the script using your preferred Roblox executor.

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/kingz7x/FlyGuiMacOs/refs/heads/main/Main.lua"))()
```

Or paste the source directly into your executor.

## Controls

| Action        | Key           |
| ------------- | ------------- |
| Move Forward  | W             |
| Move Backward | S             |
| Move Left     | A             |
| Move Right    | D             |
| Fly Up        | Space         |
| Fly Down      | C / Left Ctrl |
| Toggle UI     | Alt           |

## Notes

* Fly and NoClip functionality may not work in games with strong anti-cheat systems.
* Use responsibly.
* Abuse may result in moderation actions depending on the game.
